# up2d8 - React Native Mobile App

A React Native mobile application for personal news digests and information management.

## Project Structure

This repository contains the React Native mobile application located in the `up2d8ReactNative/` directory.

## Getting Started

### Prerequisites

- Node.js >= 20
- React Native development environment (Xcode for iOS, Android Studio for Android)
- For detailed setup instructions, see `up2d8ReactNative/README.md`

### Installation

```bash
cd up2d8ReactNative
npm install
```

### Running the App

**iOS:**
```bash
npm run ios
```

**Android:**
```bash
npm run android
```

### Development

```bash
npm start  # Start Metro bundler
```

## Documentation

- **React Native App**: See `up2d8ReactNative/README.md` for detailed mobile app documentation
- **Xcode Setup**: See `up2d8ReactNative/Xcode_Run_Instructions.md` for iOS-specific setup

## Tech Stack

- **Mobile**: React Native 0.82.1
- **UI**: React Native components with custom theming
- **Navigation**: React Navigation 7.x
- **Languages**: TypeScript

## Project Status

This is the mobile application for the up2d8 platform. The React Native app is the primary focus of this repository.
